Prerequisite

jPopulator requires a Java 1.5+ runtime.

Build jPopulator

To build jPopulator from sources, you need to have git and maven installed and set up.

Please follow these instructions :

  • $>git clone https://github.com/benas/jPopulator.git

  • $>mvn package

  • jPopulator core jar jpopulator-${version}.jar will be generated in the target folder

Use jPopulator

jPopulator have two dependencies : commons-beanutils and commons-lang. You have to simply add these two jars in addition to the jpopulator-${version}.jar to your application's classpath.

If you use maven, you should add the following dependency to your pom.xml :

<dependency>
    <groupId>io.github.benas</groupId>
    <artifactId>jpopulator</artifactId>
    <version>${version}</version>
</dependency>
